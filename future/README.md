# future/ — exhibits & fixtures for speculation

Mockups, style studies, and design fixtures for features that are still being imagined.
**Nothing in here is product code** — nothing under `src/` may import from it, and nothing
here follows the app's module rules (these files are deliberately self-contained, plain-HTML,
double-click-to-open). Some fixtures are throwaway; others are **retained as living design
references** — notably the [ui_skin/](ui_skin/) **instrument exhibits**, which stay as the
permanent visual spec for the control skin (even after it migrates into the live app) and
will keep receiving attention. Retention is per-fixture; see each entry below.

| Directory | What it explores |
|---|---|
| [ui_skin/](ui_skin/) | Control-skin mockups (future_directions §13) — mockup phase **COMPLETE**: the derivation (Round-1 style studies → Round-5 composite, *historical*) plus a locked **exhibit for all 7 instruments** (vesperia / zindel / wendelhorn / tervik / nayumi / boshwick / padlington). **The instrument exhibits are RETAINED permanently** (for now) as the living visual spec for the control skin — kept even past migration into the live app, and expected to receive further attention. Integration handoff: notes_and_status.md → "the control-skin program". Open `ui_skin/index.html`. |
