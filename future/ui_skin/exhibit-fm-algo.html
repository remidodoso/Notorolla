Exhibit — FM Algorithm Selector v1.1

A backlit-LCD algorithm display + rotary selector for Tervik's Algorithm — a lit amber/green glass with dark graphics (positive LCD, FS1R-style) showing the FM operator graph (numbered boxes + connection lines + carrier output bus, TX81Z/DX-style), and a machined selector knob below. Turn a knob (drag / wheel / click-to-cycle) through the four real 3-op routings — 1 Stack · 2 Y · 3 Pair · 4 Parallel. v1.1: uniform leg length, graph clear of the caption, knob spaced off the glass. Meta tunes the shared look.
