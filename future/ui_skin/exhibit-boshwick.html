Exhibit — Boshwick 1.0

The percussion outlier — a monotimbral drum synth whose Type rotary swaps the whole topology, so most knobs go inert per type: turn Type and watch Pitch Env light only for Kick/Tom and Snap only for Snare (the live inert-dimming stress test). Nine types is past the rotary's label range, so Type shows its value in a readout window. The tone-shaping (Tone · Snap) takes the green filter-role slot (like Zindel's Motion); no LFO/Effects → rust + blue gaps.
