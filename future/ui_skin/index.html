Control-skin mockups

The control skin (future_directions §13). Self-contained fixtures, wired to nothing but life — every control drags; lamps light; the meter runs. The instrument exhibits are the retained, living design spec (kept even past migration into the app); the Round-1→5 studies below are the historical derivation.

Control mockups

Control · FM Algorithm Selector 1.1

A new control type for Tervik's Algorithm (the deferred op-diagram idea): a backlit LCD — a lit amber/green glass with dark graphics (positive LCD, FS1R-style) — showing the classic FM operator graph (numbered boxes + connection lines + carrier output bus, TX81Z-style) with a machined rotary selector below. Uniform leg length, graph clear of the caption, knob spaced off the glass. Three variations (outline / carriers-filled + windowed knob / CRT scanlines); meta tunes tint / brightness / saturation / glow / size.

Instrument exhibits

Exhibit · Boshwick 1.0

The percussion outlier — the Type rotary swaps the whole topology, so most knobs go inert per type (turn Type: Pitch Env lights for Kick/Tom, Snap for Snare — the live inert-dimming stress test). Nine types → Type shows its value in a readout window (no radial labels). Tone-shaping (Tone·Snap) takes the green filter-role slot; no LFO/Effects → rust + blue gaps.

Exhibit · Nayumi 1.0

The richest voice — source–filter vocal synthesis read by role: Vibrato = the rust LFO, the glottal source (Voice + Breath) = the Oscillator, and the vowel formant bank IS the Filter (Vowel 5-way rotary · Size · Resonance · Soprano). Size uses the amber detent tick at its off-centre neutral. No Effects → blue gap.

Exhibit · Wendelhorn 1.0

The brass supersaw ensemble — the first panel that fills all five role hues (no gaps) and the strongest shared-cluster showcase: Ensemble = the rust LFO (pitch-LFO chorus), Pitch Atk/Time = Padlington's Oscillator sub-cluster (bipolar, amber detent tick), Lowpass = Vesperia's cluster, Stereo = a blue Effects box like Padlington's Width. Only the LFO is instrument-specific.

Exhibit · Vesperia 1.0

The reference voice and the plainest panel — the three-cluster baseline every other instrument reads against. All vertical sliders: Oscillator [Timbre — bipolar, centred detent] → Filter [Lowpass] → Envelope [Amplitude]. No LFO/Effects → the role-hue spectrum shows its gaps. Dblclick to type; wheel = coarse, tilt = fine.

Exhibit · Zindel 1.0

The drawbar additive organ — the drawbar widget in two versions (meta Drawbars): plain (a neat group of ordinary sliders) or tabs (white/black numbered pull-tabs on chrome stems, 0–8 registration, 9-position clicks, packed as a manual). up = louder on both. Acceleration takes the green filter-role slot (the "filter substitute"; band tab "Motion"); no LFO/Effects → spectrum gaps. Spread's zero sits off-center (detent at the semantic zero).

Exhibit · Tervik 1.2

The 3-op FM instrument with envelopes extracted into a separate Envelope section (Env 1·2·3); Follow replaced by 1 → 2 / 1 → 3 copy buttons (target flashes); Coarse = stepped slider (snaps to exact ratios). 1.2: the Algorithm control is now a backlit-LCD picker — a lit amber/green glass with the FM operator graph (TX81Z-style, dark-on-lit) + a plain rotary encoder (unlimited turn, wraps 1→2→3→4→1), slotted left of Fdbk. No Filter/Effects → the role-hue spectrum shows its gaps.

Exhibit · Padlington 1.3

The composite skin on a real instrument panel — the production-target look: canonical group order (Oscillator → Filter → Envelope → Effects), hue = role, boxed fieldset groups with centered legend tabs, label-only subgroup chrome (every subgroup labeled — Amplitude, Lowpass), live inert dimming (turn Source). Speculative elements removed.

E · Round 5 Composite

Ticks restored (R4 bug); per-slider values back, unit-less (5ch windows fit the column — units in the rollover); wide Jupiter-8 caps; rotary switches with position labels (text + glyph demos); black-on-color header bands — per-group tasteful palette vs single hue (meta). Round 1 — the four directions (reference)

A · Stage Piano

The RD-2000 reading — charcoal, crisp white legends, LED-indexed sliders (the value shown luminously, riding the cap), sparse red accents.

B · Color-Coded

The Roland-colors reading — each cluster carries a functional accent hue (envelope green, filter orange, pitch blue…) in its chip, track fill, and readouts. Color says which system you're touching.

C · Rack Mixer

The TotalMix reading — flat and crisp, 1px borders, real tick scales, LCD-style readouts, denser. Legibility from contrast and scale marks, not glow.

D · Machined

The literal top-down view of a real thing — knurled knobs, grip-grooved fader caps in recessed slots, a brushed panel with corner screws, engraved legends.

How to review: open each skin and drag everything — sliders, drawbars, knobs (vertical drag; the ± knobs snap to their center detent, double-click resets), toggles. Use the meta strip at the top to tune the skin live: Panel = background lightness (the "medium-dark gray, not black" dial), Accent = hue (in B it shifts the whole palette), Size = control scale, Glow = LED/lamp intensity. React per-element — "A's knobs, C's faders, B's colors" is exactly the kind of feedback Round 2 composites from, and meta-strip positions you like are spec numbers (just report them).
